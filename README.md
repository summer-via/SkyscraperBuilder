# 摩天大楼建造游戏

这是一个简单的摩天大楼建造小游戏，玩家需要控制楼体方块从顶部上下落，尽可能保证楼体掉落的位置精准，误差太大就导致楼体倒塌。

## 游戏功能

1. **完整的游戏流程**：开始、失败、得分、重新开始
2. **物理效果**：楼体的平衡，模块的左右摇晃
3. **简单的操作**：玩家只需要控制模块的释放
4. **音效系统**：放置需要有对应的音效，完全对齐会有特殊奖励音效，得分也更高
5. **难度递增**：随着楼体的增高，建筑自身摇晃幅度也会变大，以此来逐渐增加难度

## 游戏设置

1. **创建方块预制体**：
   - 在Unity中创建一个2D Sprite对象
   - 添加Rigidbody2D组件
   - 添加BoxCollider2D组件
   - 添加Block.cs脚本
   - 将其保存为预制体，命名为Block

2. **场景设置**：
   - 创建一个空对象，命名为BuildingParent，作为所有方块的父对象
   - 创建UI文本对象，命名为ScoreText，用于显示分数
   - 创建UI文本对象，命名为GameOverText，用于显示游戏结束信息
   - 创建UI按钮对象，命名为RestartButton，用于重新开始游戏
   - 添加三个AudioSource组件，分别用于放置音效、完美对齐音效和游戏结束音效

3. **Game.cs脚本设置**：
   - 将Game.cs脚本添加到主相机或空对象上
   - 在Inspector面板中，将相应的对象拖放到对应的字段中

## 游戏玩法

1. 点击鼠标左键释放当前方块
2. 尽可能让方块与下方方块对齐
3. 完全对齐会获得额外分数和特殊音效
4. 随着楼层增加，建筑会变得越来越摇晃，增加游戏难度
5. 如果方块掉落或建筑倾斜过度，游戏结束
6. 点击重新开始按钮可以重新开始游戏

## 技术实现

- **物理系统**：使用Unity的2D物理引擎实现方块的下落和碰撞
- **音效系统**：使用AudioSource播放不同的音效
- **UI系统**：使用Unity的UI系统显示分数和游戏结束信息
- **游戏逻辑**：在Game.cs中实现游戏的核心逻辑，包括方块移动、释放、对齐检测和游戏结束判断
